## Features

<CardGrid stagger>
	<Card title="Full Customization" icon="pencil">
		Customize both the thumb and track with your own composables. Create unique dial designs for any use case.
	</Card>
	<Card title="Flexible Arcs" icon="setting">
		Support for full circles, semi-circles, or any arc range you need. Define start and sweep angles easily.
	</Card>
	<Card title="Step Snapping" icon="list-format">
		Enable discrete step values for precise selection. Perfect for settings, timers, and mode selectors.
	</Card>
	<Card title="Multiplatform" icon="rocket">
		Built with Compose Multiplatform - works on Android, iOS, Desktop, and Web.
	</Card>
</CardGrid>

## Quick Example

```kotlin
var degree by remember { mutableFloatStateOf(180f) }

Dial(
    degree = degree,
    onDegreeChanged = { degree = it },
    modifier = Modifier.size(200.dp),
    startDegrees = 0f,
    sweepDegrees = 360f,
)
```
